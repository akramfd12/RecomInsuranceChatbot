🛡️ Insura Insurance Chatbot

Insura Insurance Chatbot is an AI-powered multi-agent insurance chatbot built with Python, Streamlit, and LangGraph (Swarm).
It provides insurance product information, policy details, and claim assistance, enhanced with RAG (Retrieval-Augmented Generation) using Qdrant Vector Database.

🚀 Features

🤖 Multi-Agent Architecture

Product Agent → Insurance products & premium calculation

Policy Agent → Policy / membership information

Claim Agent → Claim procedures & claim status

🔄 Automatic Agent Handoff (Swarm Orchestration)

🧠 RAG for Insurance Products (Qdrant)

🗄️ SQLite Database for Policy & Claim Data

📧 Policy Information via Email

🖥️ Streamlit UI with Sidebar Navigation

OPENAI API for NLP
